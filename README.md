# InkluSave Frontend

InkluSave is a financial inclusion platform designed to bridge the gap between underserved Nigerians and essential financial services.

This repository contains the frontend application built with React, providing users with access to:

- Digital Wallets
- Money Transfers
- AI-Powered Loan Eligibility Checks
- Community Savings Groups (Ajo/Adashe)
- Financial Literacy Resources
- User Authentication and Account Management
- Mobile-First Responsive Experience

The platform aims to empower small traders, farmers, women entrepreneurs, and low-income earners by providing accessible, secure, and inclusive financial tools.

## Tech Stack

- React
- React Router
- Axios
- Context API 
- REST APIs

## Vision

To make financial services accessible to every Nigerian, regardless of income level, location, or device type.
